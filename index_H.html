OpenModal is a measurement, analysis and visualisation software for structural dynamics analysis. It is an open source software made with Python.

Geometry Builder

Assemble the geometry of the measured object using primitives such as nodes, lines and triangles. Alternatively, import geometry from excel or uff.

Data Acquisition

Acquire data directly or import it from uff. Currently only NI hardware is supported.

Modal Analysis

Perform modal analysis using state-of-the-art methods. Reconstruct response from the calculated model.

Visualization

Visualize and animate results.

Available In 2016!
