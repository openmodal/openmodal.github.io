What is OpenModal?

The mission of the OpenModal project is to create an open modal-analysis framework that will accelerate the implementation of research ideas. The framework covers basic steps of geometry definition, data acquisition, identification and animation. It is based on the open Python ecosystem end enables the use of the same code for scientific research as well as for the final implementation.
