OpenModal is an open source
experimental modal analysis software written in Python.

OpenModal is open source experimental modal analysis software written in Python. The intuitive user interface can be used to define nodes and elements, perform data acquisition, modal identification analysis and interactive visualization of results. It promotes open science and collaboration on open source projects.

FEATURES

The complete experimental modal analysis procedure.

Define geometry

Geometry builder lets you create your models from nodes with specified coordinates and coordinate systems. It supports 2D and 3D primitives such as lines, planes, cylinders and boxes of different shapes. Large node patterns can be generated automatically by using the geometry wizard.

Measure signals

Acquire data in experiments or import it from uff. Operational and experimental modal analysis are supported through the options of different excitations signals, such as impulse or random. National Instruments hardware is the only one supported at the moment.

Modal analysis

Modal parameters are identified by a process of curve fitting that is used to estimate the poles (natural frequencies and damping ratios). The LSCF and LSCE methods are available.

Visualization

Measured and analyzed data can be visualized and animated giving insight into dynamics of the structure. Each mode can be visualized as well as measured data, based on the frequency-response spectrum.

OPEN LIBRARIES

Available through pip and on Github.

The OpenModal project spawned the development of Python libraries that may be useful for other projects related to signal processing, structural analysis, measurement and others. We will continue to provide standalone libraries when parts of OpenModal grow mature enough. So far, the following Python libraries have been made available on PyPI and on Gihtub.

pyuff

Defines a Python object for manipulating with the UFF (Universal File Format) files. Implements multiple fields for storing geometry, coordinate systems, measured frequency-response functions and experimental modal analysis results.

Github / PyPI

lvm_read

Implements a parser for reading text-type LabView Measurement files. It reads the header data and supports files that are split into multiple sections.

Github / PyPI

pyFRF

Frequency-response functions (common in analysis of structural dynamics), can be defined using the pyFRF object. Different window and averaging options are available. The library can be used with impulse as well as random excitation.

Github / PyPI
